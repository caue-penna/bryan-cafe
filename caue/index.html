Biography

I am Caue Penna, a web developer currently studying for the Diploma of Information Technology (Web Development) at the Australian College of Business Intelligence (ACBI). I designed and built this Bryan's Café website end-to-end as part of the combined ICTWEB441 / ICTWEB518 assessment, applying client-side scripting (JavaScript) together with structured data (XML and DTD) to create a dynamic, responsive and accessible web experience.

My background spans strategy and technology, and over the last few years I have grown a strong interest in front-end engineering, design systems and the practical engineering behind delightful user interfaces. I enjoy turning real-world business requirements into clean, maintainable code — from the initial wireframe through to deployment, testing and iteration.

For this project I authored the HTML structure, the responsive CSS design system, the JavaScript that loads and renders the menu and branch data from external XML documents, and the corresponding DTDs that validate them. The whole site is hosted publicly so that it can be reviewed live, on both laptop and mobile devices.

Outside of class, I enjoy continuing to develop side projects and reading about software craft, accessibility and modern web standards. Thank you for taking the time to review my work on this assessment.
